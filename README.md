# BetterStatsPage

**Version:** 1.0.2  
**Author:** the-xorcist  
**Release Date:** 2026-02-19

## Description

BetterStatsPage is a BepInEx mod for Erenshor that enhances the player stats interface with a comprehensive tabbed UI. View detailed character statistics, faction reputation, and more in an elegant, draggable window with smooth animations and automatic updates.

## Features

- **Tabbed Interface** - Switch between Stats and Reputation views with a clean tab system
- **Comprehensive Stats Display**:
  - Level and Ascension Level tracking
  - Experience progress
  - All attributes (Str, Dex, End, Agi, Int, Wis, Cha) with item bonuses shown separately
  - Crit Chance and Dodge Chance
  - Physical Attack, Spell Damage Bonus, Healing Bonus
  - Life Steal and Damage Shield
  - Attack Speed, Move Speed, and Resonance Proc Chance
- **Reputation Tab** - View all faction standings in one place
- **Draggable Window** - Reposition the panel anywhere on screen using the drag handle
- **Auto-Update** - Stats refresh automatically when they change
- **Smooth Animations** - Fade in/out transitions for a polished experience
- **Extensible API** - Other mods can register custom tabs via TabRegistry

## Installation

1. Install BepInEx for Erenshor (if not already installed)
2. Copy `BetterStatsPage.dll` to your `BepInEx/plugins` folder
3. Launch the game
